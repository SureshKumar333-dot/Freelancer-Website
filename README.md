# ✨ Slide Animation Portfolio

A modern, animated portfolio website built using **HTML, CSS, and JavaScript**.
This project features smooth animations, custom cursor effects, particle backgrounds, and responsive design.

---

## 🚀 Live Demo

👉 https://sureshkumar333-dot.github.io/Slide-animation/

---

## 📌 Features

* 🎯 Custom animated cursor
* 🌌 Particle background animation
* 📜 Smooth scrolling navigation
* 🎨 Elegant UI with gold & dark theme
* 📊 Animated counters
* 🧩 Skills filtering system
* 💬 Contact form (UI only)
* 📱 Fully responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (Advanced animations & variables)
* JavaScript (Vanilla JS)

## ⚙️ How to Run Locally

1. Clone the repository:

```
git clone https://github.com/SureshKumar333-dot/Slide-animation.git
```

2. Open the folder in VS Code

3. Run using Live Server OR open `index.html` in browser

---

## 🌐 Deploy on GitHub Pages

1. Go to **Settings**
2. Click **Pages**
3. Select branch: `main`
4. Save

Your site will be live at:

```
https://sureshkumar333-dot.github.io/Slide-animation/
```

---

## ⚠️ Important Notes

* Make sure all files are named correctly:

  * `index.html`
  * `style.css`
  * `script.js`

* Ensure IDs used in JavaScript exist in HTML:

  * `cursor`
  * `cursor-ring`
  * `particle-canvas`
  * `navbar`
