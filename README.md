A modern, responsive portfolio website for a freelance UI/UX Designer & Web Developer. It showcases skills, selected work, client testimonials, and a contact section.

## 🚀 Features

- **Hero Section:** A compelling introduction with a tagline and call-to-action.
- **About Me:** Brief bio highlighting the "design and code" full-stack approach.
- **Skills & Services:** Showcases key services (UI Design, UX Research, Prototyping) and a dynamic list of technologies (Figma, React, Next.js, etc.).
- **Portfolio Grid:** Displays six selected projects with descriptions and tags for design, development, or full-stack work.
- **Testimonials:** Carousel or grid of client reviews with star ratings.
- **Contact Section:** A contact area with an email address, location, and a simulated "Message Received" confirmation.
- **Responsive Design:** The layout is designed to work seamlessly on desktops, tablets, and mobile devices.

🎨 Customization
Personal Information: Update the name, bio, email, and location in index.html.

Projects: Replace the project images, titles, descriptions, and links in the portfolio section.

Skills: Modify the list of technologies in the "Skills & Services" section.

Styling: Edit css/style.css to change colors, fonts, and layout.

Colors: The primary color and accents can be easily modified in the CSS variables.

📱 Responsive Design
The website uses a mobile-first approach. It is fully responsive and tested on:

Mobile devices (320px and up)

Tablets (768px and up)

Desktops (1024px and up)

Large screens (1440px and up)
